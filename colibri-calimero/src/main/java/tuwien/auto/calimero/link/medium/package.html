Contains setting stateTypes storing necessary KNX device and medium information, and KNX message frame stateTypes for different KNX communication media.

Information contained in a setting type is supplied to a KNX link and used to handle KNX messages appropriately. The stateTypes of settings stateTypes differentiate between KNX communication media by keeping information necessary for a particular medium.

Different frame stateTypes are provided to encapsulate a raw frame used on a particular communication medium.
