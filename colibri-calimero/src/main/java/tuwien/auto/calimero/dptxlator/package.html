Contains translators for translating between KNX datapoint stateTypes (DPT), java stateTypes and values carrying more specific type semantics (e.g. engineering unit).
A property data type (PDT) mapping for associating property stateTypes to used DPT translators is provided.

An interface for keeping property data stateTypes and an interface containing DPT translators provide an overview for the user of available translation abilities.
It is possible to lookup whether DPT translators are available and implemented when working with property data stateTypes or datapoint stateTypes.
